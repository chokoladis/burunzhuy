# Проект - аукцион идей на nodejs(nestjs)
___
## Техническая часть
### Сущности:
- User
> Поля: Email, Телефон, Имя, Фамилия, фото, текст описания,
> активность,
> Группа (продавец, покупатель, менеджер или администратор),
> дата создания и изменения
- Group
> Поля: Название, Символьный код
- Idea
> Поля: Заголовок, короткое описание, превью, полное описание, файлы, минимальная стоимость входа, полная цена по которой можно выкупить сразу
- Auction
> Поля: Идея, покупатель, статус, дата время создания и окончания

### Дополнительно
- проверка пользователей по куки, браузеру + размеру окна, ip, номеру телефона
- Оплата блокируется на 7 дней
- Проверка доступов по пренадлежности к группе
- Проверка наличия контактов и опред. ссылок в: user (описание), idea (все поля)
- Предусмотрена подписка уведомлений по категориям идей

___
## Процесс
1) Покупатель открывает аукцион по входной цене
2) Открывается диалог по аукциону и торги,
3) В аукцион могут заходить другие покупатели с ценой выше последнего на 5-10% или указать свою

#### Подробности процесса
1) В самом аукционе сообщения покупателей и продавца, системные сообщения, кнопки «предложить цену», «отказаться» и отсчет.
2) Первый взнос снимается у покупателя только в случае, если аукцион закончился без дополнительных ставок
3) Продавец может заранее закончить аукцион, но не ранее чем через час. Стандартно окончание через 12ч

#### Что конкретно получает участник и покупатель?
> Участник получает возможность делать ставки, покуптель получает - полное описание и файлы из "идеи"
> <br>* Если покупателя не устроила покупка, он может оспорить такую покупку подав соответсвующую заявку в течении суток, заявка может обрабатываться до 7 дней
